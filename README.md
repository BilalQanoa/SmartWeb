# SmartWeb - AI-Powered Academic SaaS Platform

SmartWeb (SmartApp) is a modern web application built with **Django** and **Tailwind CSS**. It enables university professors, researchers, and academic labs to instantly generate and deploy professional, minimalist portfolio websites. The core feature is an integrated **AI Writer Engine** that translates raw academic notes into polished biographies, formats publications, and structures interactive course syllabi without requiring any coding knowledge.

---

## 🚀 Key Features

* **AI-Powered Prose:** Converts bullet points into high-quality, tenure-grade academic biographies.
* **Publications Manager:** Automated formatting and filtering of research papers into structured APA/IEEE citation blocks.
* **Interactive Course Hub:** Dynamic syllabus builder and schedule organizer for university students.
* **Minimalist SaaS Layouts:** Premium responsive templates inspired by modern tech interfaces (Stripe/Linear aesthetics).
* **Robust Backend:** Secure user authentication, profile tracking, and efficient database relations.

---

## 🛠️ Tech Stack

* **Backend:** Python 3.11+, Django 5.0+
* **Frontend:** Tailwind CSS, JavaScript (Fetch API / Async)
* **AI Integration:** OpenAI API / ChatCompletion (GPT models)
* **Database:** SQLite (Development) / PostgreSQL (Production)
